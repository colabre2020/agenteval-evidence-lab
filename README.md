# AgentEval Evidence Lab

Evidence repository for the paper **"Evaluating LLM Agents in Real-World Software Engineering Workflows"**.

## Contents
- `artifacts/`: JSON summaries used in manuscript tables and analysis.
- `notebooks/`: Colab-ready notebook for evidence generation.
- `tables/`: Input CSVs and generated table artifacts.

## Reproducibility
1. Open `notebooks/evidence_pipeline_colab.ipynb` in Google Colab or Jupyter.
2. Use `tables/eval_runs.csv` as the canonical pilot run input.
3. Regenerate metrics and table rows from the notebook.
